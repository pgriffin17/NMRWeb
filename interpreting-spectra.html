Interpreting NMR Spectra
There are four basic aspects
of a 1H NMR spectrum.
  1. Number of peaks in a spectrum:
    Symmetry
  2. Location of peaks in a spectrum:
    Chemical Shift
  3. Size of the peak in the spectrum:
    Integration
  4. Appearance of the peak
    in the spectrum: Splitting
Symmetry
Number of Peaks
With NMR spectroscopy, every peak corresponds to a unique type of the atom being analyzed. Having four peaks in a 13C NMR spectrum doesn’t necessarily mean the compound has four carbons. The compound might have more than four carbons if there are two carbons that have the exact same chemical environment. If there is symmetry in the molecule, meaning if the carbons have exactly the same chemical environment, those carbons will all be one peak. All that can knows is that with four peaks, a molecule has four unique types of carbon.

For 1H NMR spectroscopy, most often every hydrogen on a particular carbon will be identical to the others on the same carbon. Since there are often multiple H’s on the same carbon, many peaks in the proton NMR spectrum will end up representing more than one H. Again, the information about number of peaks is that the number of peaks corresponds to the number of unique proton environments.
